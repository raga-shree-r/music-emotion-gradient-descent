# Music Emotion Prediction using Gradient Descent

## Problem
This project aims to predict the emotional response to music based on features such as genre and user rating.

## Approach
- Data preprocessing and cleaning
- Feature selection (Genre, Rating)
- Encoding categorical variables
- Implementing gradient descent from scratch
- Minimizing Mean Squared Error (MSE)

## Results
- Loss decreases over iterations, showing successful optimization
- Model demonstrates how gradient descent can learn patterns from data

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
